## Eatery cover photos (seed / demo)

Loaded from remote URLs at runtime — **not bundled**. Generic food photography only; no chain logos or scraped venue photos.

| Theme | Source | License / notes |

| Biryani, nihari, desi plates | [Unsplash](https://unsplash.com) | Unsplash License — category stand-ins (e.g. Student Biryani, Waris Nihari) |
| BBQ / seekh kabab | Unsplash | Generic grill photography — not venue-specific |
| Café / brunch | Unsplash | Generic café interiors — Xander's, Espresso, etc. |
| Seafood | Unsplash | Generic seafood plates — Do Darya grill, Sea View fish |
| Street food / chai | Unsplash | Generic street-food and chai — Burns Road, Boat Basin |

When no venue-specific licensed photo exists, we use the closest **food category** image and document it here — never mislabeled as an official restaurant logo or interior.
